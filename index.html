What is DogeCoin?

A cryptocurrency created in 2013 as a joke by Jackson Palmer and Billy Markus. Doge is a meme which the wanted to change into a cryptocurrency. Now it has a market cao of over $1 billion and Elon Musk is supporting it. It was originally designed as an alternative to Bitcoin. Today DogeCoin is mostly used as a tipping service for posts and contributions which need recognition.

Fun Facts: There had been no technical updates in the code of Dogecoin since 2015. Elon Musk calls it a "cryptocurrency of people".
